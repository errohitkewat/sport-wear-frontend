# SportWear Frontend

A modern full-stack sportswear e-commerce frontend built with React, TypeScript, Tailwind CSS, Framer Motion, and React Router.

## Features

- Modern responsive sportswear store UI
- Dynamic product listing from backend API
- Product details page
- Men and Women collection pages
- Cart functionality
- Checkout page
- User authentication flow
- Protected routes
- Admin dashboard UI
- Animated modern design

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router

- Home
